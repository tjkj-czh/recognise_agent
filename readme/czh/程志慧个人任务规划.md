## 核心Skills详细设计

### Skill 1：地类标签映射Skill
- **用途**：将模型输出的原始分类标签映射为业务标准地类标签
- **适用前提**：已有分类体系表和标签映射规则
- **输入格式**：模型原始输出标签 + 分类体系版本号
- **输出格式**：业务标准地类标签 + 标签来源说明
- **失败条件**：遇到映射表中未定义的标签
- **人工接管条件**：新标签无法自动归类时，标记为"待人工确认"

### Skill 2：图斑主导类别判定Skill
- **用途**：判定图斑的主导地类类型
- **适用前提**：图斑已有像素/对象级分类结果
- **输入格式**：图斑几何 + 各类别面积占比
- **输出格式**：主导地类标签 + 面积占比 + 判定依据
- **失败条件**：无类别占比超过阈值
- **人工接管条件**：前两名类别占比接近（差距<5%），标记为"混合待判"

### Skill 3：分类结果聚合Skill
- **用途**：将像素/对象级分类结果聚合到图斑层级，形成面积、占比、主导类型等统计
- **适用前提**：图斑边界和分类结果已对齐
- **输入格式**：图斑矢量 + 像素/对象分类结果
- **输出格式**：图斑级统计表（各类别面积、占比、主导类型、碎片化程度）
- **失败条件**：图斑与分类结果无法空间对齐
- **人工接管条件**：聚合后碎片化程度超过阈值，标记为"分割质量存疑"

### Skill 4：低置信区识别Skill
- **用途**：识别低置信度区域并标记，避免低质量判读流入下游
- **适用前提**：分类结果附带有置信度信息
- **输入格式**：图斑分类结果 + 像素置信度
- **输出格式**：低置信区标记 + 原因说明 + 建议处理方式
- **失败条件**：无置信度信息可用
- **人工接管条件**：大面积低置信区域，标记为"建议人工复核"

### Skill 5：混合地类提示Skill
- **用途**：识别和提示混合地类图斑，给出各类占比和混合特征
- **适用前提**：图斑内存在多种地类且无单一主导类型
- **输入格式**：图斑级分类统计结果
- **输出格式**：混合类型标签 + 各类占比 + 混合特征说明 + 下游提示
- **失败条件**：无法确定混合类型组成
- **人工接管条件**：混合类型复杂度超阈值，标记为"需人工判读"

## 关键区分："识别结果" vs "业务结论"

这是答辩核心要点，必须在Agent设计中严格体现：

| 维度 | 识别结果（我的输出） | 业务结论（王新林的输出） |
|------|---------------------|------------------------|
| 性质 | 技术性判读 | 业务性判断 |
| 示例 | "该图斑耕地占比65%，置信度0.87" | "该图斑涉及耕地保护关注区，建议复核" |
| 边界 | 只说"是什么" | 说"意味着什么" |
| 风险 | 不能把识别结果直接当业务结论 | — |

**我的Agent绝不输出**：违规判断、合规结论、处罚建议等业务定性内容。
**我的Agent一定输出**：地类标签、置信度、混合类型提示、难判标记、低置信区说明。
